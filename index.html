Monte Carlo Ray Tracer (ongoing)

A Monte Carlo ray tracer developed as part of the course Advanced Global Illumination. The renderer will include transparent, perfectly reflecting objects, but also Lambertian reflections. Additionally, a simple photon mapping scheme will be implemented. The renderer is being written in C++.

More details can be found in the upcoming report.

Create 2D character sprites with deep learning (ongoing)

Using an image-2-image CycleGAN translation model (U-GAT-IT) to create new 2D character sprites for video games from real life images of people. The project is executed as part of the course Artificial intelligence for interactive media.

More details can be found in the upcoming report.
